# Description

This project is a payroll system created for our Computer Programming 1 (MO-IT101) course. The program calculates an employee’s weekly salary based on the total hours they worked. The system reads the employee’s log-in and log-out records to compute their working hours. It also considers the company’s office hours and allows a 10-minute grace period for late arrivals.

Government deductions such as SSS, PhilHealth, Pag-IBIG, and withholding tax are originally based on monthly rates. The program converts these deductions into weekly amounts so the employee’s net weekly salary can be calculated.

All results are shown in the program, including employee details, hours worked, gross salary, deductions, and net pay.

## Usage

**Employee Lookup:** Enter employee credentials to fetch details.
**Payroll Staff Access:** Use payroll staff credentials for administrative lookup.
**Hours Worked Calculation:** Input time logs to compute total hours.
**Gross & Net Salary:** System automatically calculates salary, deductions, and contributions (SSS, PhilHealth, Pag-IBIG, Tax).
